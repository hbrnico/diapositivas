MÉTODOS DE ORDENAMIENTO

SELECCIÓN, INSERCIÓN, BURBUJA Y MERGESORT

Selección Directa

Busca el elemento mínimo de la lista y lo intercambia con el primero. Luego busca el siguiente mínimo en el resto y lo intercambia con el segundo, y así sucesivamente.

Características:

  • Fácil de implementar.
  • Complejidad temporal: O(n²).
  • Realiza pocos intercambios físicos de elementos.

void seleccion(int vec[], const int tl) {
  int i, j, min;
  for (i = 0; i < tl; i++) {
    min = i;
    for (j = i + 1; j < tl; j++) 
      if (vec[j] < vec[min]) min = j;
    intercambio(vec[i], vec[min]);
  }
}

Inserción Directa

Ordena insertando cada elemento en su posición correcta respecto a los elementos que ya han sido ordenados previamente (similar a cómo se ordenan cartas en la mano).

Características:

  • Eficiente para listas pequeñas o semiordenadas.
  • Complejidad temporal: O(n²) en peor caso, O(n) en el mejor.
  • Algoritmo de ordenamiento estable.

void insercion(int vec[], const int tl) {
  int i, j;
  for (i = 1; i < tl; i++) {
    j = i;
    while ((j > 0) && (vec[j] < vec[j - 1])) {
      intercambio(vec[j], vec[j - 1]);
      j = j - 1;
    }
  }
}

Ordenamiento Burbuja

Compara elementos adyacentes y los intercambia si están en el orden incorrecto. El elemento más grande "flota" hacia el final en cada pasada.

Características:

  • Conceptualmente el más simple.
  • Complejidad temporal: O(n²).
  • Realiza una gran cantidad de escrituras en memoria debido a los múltiples intercambios.

void burbuja(int vec[], const int tl) {
  int pasada, k;
  for (pasada = 1; pasada < tl; pasada++) {
    for (k = 0; k < tl - 1; k++)
      if (vec[k] > vec[k + 1])
        intercambio(vec[k], vec[k + 1]);
  }
}

Marge Sort

Merge Sort: Divide y Vencerás

Divide recursivamente el arreglo en mitades iguales hasta tener arreglos de tamaño 1. Luego, mezcla ordenadamente las mitades adyacentes.

Características:

  • Complejidad temporal garantizada: O(n log n).
  • Usa memoria auxiliar temporal durante la mezcla.
  • Altamente eficiente para arreglos grandes.

void mergesort(int v[], int inicio, int final) {
    if (final - inicio == 0) return;
    else {
        mergesort(v, inicio, (inicio + final) / 2);
        mergesort(v, (inicio + final) / 2 + 1, final);
        merge(v, inicio, (inicio + final) / 2, (inicio + final) / 2 + 1, final);
    }
}

void merge(int v[], int inicio1, int final1, int inicio2, int final2) {
    int i = inicio1, j = inicio2, k = 0;
    int c[100]; // Memoria temporal

    while (i <= final1 && j <= final2) {
        if (v[i] < v[j]) c[k++] = v[i++];
        else c[k++] = v[j++];
    }

    while (i <= final1) c[k++] = v[i++];
    while (j <= final2) c[k++] = v[j++];

    for (k = 0; k < final2 - inicio1 + 1; k++)
        v[inicio1 + k] = c[k];
}

Ejercicio 2: Eliminar y Ordenar

Diseñar la función elimOrdenar que recibe un arreglo desordenado A de 500 números enteros, su tamaño lógico TLA y un entero X. La función debe:

1. Eliminar múltiplos

Eliminar todos los valores del vector A que sean múltiplos de X.

2. Ordenar

Dejar el vector ordenado ascendentemente.

3. Retornar cantidad

Retornar cuántos elementos fueron eliminados del vector A.

Solución: Ejercicio 2


int elimOrdenar(int A[], int &TLA, int X) {
    int eliminados = 0;
    
    // 1. Eliminar múltiplos de X (desplazando a la izquierda)
    int i = 0;
    while (i < TLA) {
        if (A[i] % X == 0) {
            for (int j = i; j < TLA - 1; j++) {
                A[j] = A[j + 1];
            }
            TLA--;
            eliminados++;
        } else {
            i++;
        }
    }
    
    // 2. Ordenar ascendentemente (Método Selección)
    for (i = 0; i < TLA; i++) {
        int min = i;
        for (int j = i + 1; j < TLA; j++) {
            if (A[j] < A[min]) min = j;
        }
        intercambio(A[i], A[min]);
    }
    
    return eliminados;
}

Ejercicio 3: Intersección

La función Uno() recibe dos arreglos desordenados (A y B, con TB ≤ TA), sus tamaños lógicos (TA y TB) y un booleano C. La función debe:

¿Qué debe retornar?

  • Un nuevo vector V y su tamaño lógico TV.
  • Debe contener solamente los valores de B que están en A, sin repeticiones.
  • El vector V debe quedar ordenado ascendentemente si C == true (descendentemente en caso contrario).

Solución: Ejercicio 3


bool existe(int vec[], int tl, int valor) {
    for (int i = 0; i < tl; i++)
        if (vec[i] == valor) return true;
    return false;
}

void Uno(int A[], int TA, int B[], int TB, bool C, int V[], int &TV) {
    TV = 0;
    
    // 1. Filtrar elementos de B que están en A sin repeticiones
    for (int i = 0; i < TB; i++) {
        if (existe(A, TA, B[i]) && !existe(V, TV, B[i])) {
            V[TV] = B[i];
            TV++;
        }
    }
    
    // 2. Ordenar según C (Selección)
    for (int i = 0; i < TV; i++) {
        int target = i;
        for (int j = i + 1; j < TV; j++) {
            if (C) {
                if (V[j] < V[target]) target = j; // Ascendente
            } else {
                if (V[j] > V[target]) target = j; // Descendente
            }
        }
        intercambio(V[i], V[target]);
    }
}
