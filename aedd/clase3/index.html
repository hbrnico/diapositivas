Ejercicio 10 - Concatenación

Objetivo: Extraer prefijos y sufijos de longitud determinada y combinarlos.

Especificaciones:

  • Solicite el ingreso de una frase.
  • Extraiga los primeros 5 caracteres y los últimos 5 caracteres.
  • Muestre por separado ambos segmentos y luego su concatenación.

Restricción: Asuma que la cadena ingresada posee al menos 5 caracteres.

Solución: Ejercicio 10


#include <iostream>
#include <string>
using namespace std;

int main() {
    string frase;
    cout << "Entrada: ";
    cin >> frase;

    string primeros = frase.substr(0, 5);
    string ultimos  = frase.substr(frase.length() - 5, 5);

    cout << "Primeros 5: " << primeros << endl;
    cout << "Últimos 5: "  << ultimos  << endl;
    cout << "Concatenación: " << primeros + ultimos << endl;

    return 0;
}

Ejercicio 4 - Guía 4

Un postulante a un empleo realiza un test de capacitación. Se ingresa por teclado la cantidad total de preguntas realizadas y la cantidad de preguntas contestadas correctamente.

Desarrollar un programa que informe el nivel obtenido, sabiendo que:

  • Nivel máximo: Porcentaje >= 90.
  • Nivel medio: Porcentaje >= 75% y < 90%.
  • Nivel regular: Porcentaje >= 50% y < 75%.
  • Fuera de nivel: Porcentaje < 50%.

Solución: Ejercicio 4


#include <iostream>
using namespace std;

int main() {
    int total, correctas;
    cout << "Total de preguntas: ";
    cin >> total;
    cout << "Respuestas correctas: ";
    cin >> correctas;

    float porcentaje = (correctas * 100.0) / total;

    if (porcentaje >= 90)
        cout << "Nivel máximo" << endl;
    else if (porcentaje >= 75)
        cout << "Nivel medio" << endl;
    else if (porcentaje >= 50)
        cout << "Nivel regular" << endl;
    else
        cout << "Fuera de nivel" << endl;

    return 0;
}

Ejercicio 7 - Guía 4

Diseñe un algoritmo que lea un número de tres cifras y determine si es o no capicúa. Un número es capicúa si “se lee igual al derecho y al revés”.

Ejemplo:

  • Si se ingresa 383, indica que es capicúa.
  • Si se ingresa 473, indica que no es capicúa.

Solución: Ejercicio 7


#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "Ingrese un número de tres cifras: ";
    cin >> num;

    int centena = num / 100;
    int unidad  = num % 10;

    if (centena == unidad)
        cout << num << " es capicúa" << endl;
    else
        cout << num << " no es capicúa" << endl;

    return 0;
}

Ejercicio 6 - Guía 4

El cliente de un supermercado puede pagar con:

  • Tarjeta de débito ('d', 'D')
  • Billetera Santa Fe ('b', 'B')
  • Tarjeta de crédito ('c', 'C') en 1 o 3 cuotas.

Condiciones:

  • Crédito: En 1 cuota no hay recargo. En 3 cuotas hay un 10% de recargo.
  • Débito: Si supera $3000, hay un 15% de descuento. Si no, ningún cambio.
  • Billetera SF: Siempre 30% de descuento.

Solución: Ejercicio 6


#include <iostream>
using namespace std;

int main() {
    float monto, total;
    char pago;
    cout << "Monto: "; cin >> monto;
    cout << "Pago (C/D/B): "; cin >> pago;

    if (pago == 'c' || pago == 'C') {
        int cuotas;
        cout << "Cuotas (1 o 3): "; cin >> cuotas;
        if (cuotas == 3) {
            total = monto * 1.10;
            cout << "Por cuota: $" << total / 3 << endl;
        } else {
            total = monto;
        }
    } else if (pago == 'd' || pago == 'D') {
        if (monto > 3000) {
            total = monto * 0.85;
        } else {
            total = monto;
        }
    } else if (pago == 'b' || pago == 'B') {
        total = monto * 0.70;
    }
    cout << "Total: $" << total << endl;
    return 0;
}
