11340 - Etapas de la vida

Dieguito vio en su clase de biología las etapas de la vida, pero aún no las distingue muy bien. Tu tarea es ayudar a Dieguito haciendo un programa el cual, dado un número de entrada L, el cual representa una edad, te diga si dicha edad corresponde a un bebé, niño, joven, adulto o adulto de la tercera edad. Para lograr lo anterior, considera la siguiente clasificación:

  • Bebé: De 0 a 3 años, inclusive.
  • Niño: De 4 a 14 años, inclusive.
  • Joven: De 15 a 18 años, inclusive.
  • Adulto: de 19 a 65 años, inclusive.
  • Adulto de la tercera edad: de 65 años en adelante (máximo 99 años).

Ejemplo de entrada/salida:

Entrada Salida
17 JOVEN
23 ADULTO
68 ADULTO 3RA
2 BEBE

Solución: Etapas de la vida


#include <iostream>
using namespace std;

int main(int argc, char *argv[]) {
	int edad;
	cin>>edad;
	if (edad<=3)
		cout<<"BEBE"<<endl;
	else
		if (edad>=4 && edad<=14)
			cout<<"NINO"<<endl;
		else
			if (edad>=15 && edad<=18)
				cout<<"JOVEN"<<endl;
			else
				if (edad>=19 && edad<=65)
					cout<<"ADULTO"<<endl;
				else
					if (edad>65)
						cout<<"ADULTO 3RA"<<endl;
	return 0;
}

Ejemplo:

  • Ingresando 3, 8 → Martes y Agosto
  • Ingresando 12, 7 → Día inexistente.

Solución: Ejercicio 5


#include <iostream>
using namespace std;

int main() {
    int dia, mes;
    
    cout << "Dia de la semana (1-7): "; cin >> dia;
    cout << "Mes (1-12): "; cin >> mes;
    
    if (dia > 7 || dia < 1 || mes > 12 || mes < 1) {
        cout << "Dia inexistente" << endl;
    } else {
        cout << "Dia: ";
        switch(dia) {
            case 1: cout << "Lunes"; break;
            case 2: cout << "Martes"; break;
            case 3: cout << "Miercoles"; break;
            case 4: cout << "Jueves"; break;
            case 5: cout << "Viernes"; break;
            case 6: cout << "Sabado"; break;
            case 7: cout << "Domingo"; break;
        }
        
        switch(mes) {
            case 1: cout << " y Enero\\n"; break;
            case 2: cout << " y Febrero\\n"; break;
            case 3: cout << " y Marzo\\n"; break;
            case 4: cout << " y Abril\\n"; break;
            case 5: cout << " y Mayo\\n"; break;
            case 6: cout << " y Junio\\n"; break;
            case 7: cout << " y Julio\\n"; break;
            case 8: cout << " y Agosto\\n"; break;
            case 9: cout << " y Septiembre\\n"; break;
            case 10: cout << " y Octubre\\n"; break;

3843 - Asteriscos 3

Realiza un programa que reciba un entero N e imprima una pirámide de asteriscos de altura N. Vea los ejemplos de entrada y salida.


Ejemplo:

Entrada Salida
7 
      *
     ***
    *****
   *******
  *********
 ***********
*************
5 
    *
   ***
  *****
 *******
*********

Solución: 3843 - Asteriscos 3


#include <iostream>
using namespace std;
void piramide (int);
int main(int argc, char *argv[]) {
	int N;
	cin>>N;
	if (N>=1 && N<=100){
		piramide (N);
	}
	return 0;
}

void piramide(int N){
	for(int n1=1;n1<=N;n1++){
		
		for(int n2=N;n1<n2;n2--){
			cout<<" ";
		}
		
		int n3=0;
		do{
			cout<<'*';
			n3++;
		} while(n3!=n1*2-1);
		cout<<endl;
	}
}

Ejercicio 5 - Guía 5

Estás desarrollando un programa en C++ que te permitirá realizar varias operaciones con los números ingresados por el usuario. Tu objetivo es diseñar un programa interactivo que solicite al usuario ingresar números y que calcule la suma de los números pares ingresados, la cantidad de números impares y la cantidad de números negativos, hasta que se alcance el final del archivo (EOF) mediante la combinación de teclas Ctrl + Z en Windows o Ctrl + D en sistemas Unix.

Solución: Ejercicio 5


#include <iostream>
using namespace std;

int main() {
    int numero;
    int sumaPares = 0;
    int cantImpares = 0;
    int cantNegativos = 0;

    cout << "Ingrese numeros (Ctrl+Z para terminar): " << endl;
    while (cin >> numero) {
        if (numero % 2 == 0) {
            sumaPares += numero;
        } else {
            cantImpares++;
        }
        if (numero < 0) {
            cantNegativos++;
        }
    }

    cout << "Suma de pares: " << sumaPares << endl;
    cout << "Cantidad de impares: " << cantImpares << endl;
    cout << "Cantidad de negativos: " << cantNegativos << endl;

    return 0;
}

24623 - Código Enigma

La agencia de inteligencia ha interceptado una transmisión enemiga. El mensaje llegó como una sola cadena de texto compacta: sin espacios, sin signos de puntuación y sin pausas visibles. Los analistas saben que la transmisión esconde coordenadas numéricas clave, pero el método de cifrado era un misterio.

Luego de semanas de intenso trabajo, el criptógrafo jefe logró reconstruir el algoritmo de descifrado. Dejó anotadas sus conclusiones en su diario de campo antes de desaparecer misteriosamente:

"Cada letra del alfabeto inglés carga un peso según su posición en el abecedario, empezando desde cero. Es decir, la primera letra no vale nada (A=0), la segunda vale uno (B=1), y así sucesivamente hasta la última letra (Z=25), sin distinguir entre mayúsculas y minúsculas. Sin embargo, hay una letra que no actúa como las demás: no aporta peso alguno al mensaje, sino que sirve únicamente para separar la transmisión en bloques independientes: esta es la letra N (o n).

Cada bloque se resuelve sumando el peso de todas sus letras. Pero el enemigo no usa esa suma directamente: aplica un filtro final que retiene solo ciertos dígitos del resultado y descarta las cifras que están en posiciones pares (contadas de izquierda a derecha en base 1). Al terminar de procesar todos los bloques de la transmisión, nos interesa encontrar el mayor valor resultante."

Tu tarea es escribir un programa que ayude a la agencia a automatizar este descifrado y encontrar el mayor valor resultante de la transmisión.

Solución del ejercicio



#include <iostream>
#include <cmath>
using namespace std;
int main(int argc, char *argv[]) {
	int NumBloque=0, Max=0, aux, cantCifras=1, MaxDelBloque=0, armarNum=1;
	string Cadena;
	cin>>Cadena;
	for(int i=0;i<Cadena.size();i++) { 
		if(tolower(Cadena.at(i))!='n'){
			Cadena.at(i)=tolower(Cadena.at(i));
			NumBloque+=Cadena.at(i)-'a';
		}
		if(tolower(Cadena.at(i))=='n' or i==(Cadena.size()-1)){
			aux=NumBloque;
			while(aux>9){
				aux/=10;
				cantCifras++;
			}
			for(int j=cantCifras;j>0;j--) { 
				if(j%2!=0){
					MaxDelBloque+=(NumBloque%10*armarNum);
					NumBloque/=10;
					armarNum*=10;
				}else{
					NumBloque/=10;
				}
			}
			if(MaxDelBloque>Max){
				Max=MaxDelBloque;
			}
			MaxDelBloque=0;
			NumBloque=0;
			cantCifras=1;
			armarNum=1;
		}
	}
	cout<<Max;
	return 0;
}
