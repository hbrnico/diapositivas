Ejercicio 10 - Concatenación

Objetivo: Extraer prefijos y sufijos de longitud determinada y combinarlos.

Especificaciones:

  • Solicite el ingreso de una frase.
  • Extraiga los primeros 5 caracteres y los últimos 5 caracteres.
  • Muestre por separado ambos segmentos y luego su concatenación.

Restricción: Asuma que la cadena ingresada posee al menos 5 caracteres.

Solución: Ejercicio 10


#include <iostream>
#include <string>
using namespace std;

int main() {
    string frase;
    cout << "Entrada: ";
    cin >> frase;

    string primeros = frase.substr(0, 5);
    string ultimos  = frase.substr(frase.length() - 5, 5);

    cout << "Primeros 5: " << primeros << endl;
    cout << "Últimos 5: "  << ultimos  << endl;
    cout << "Concatenación: " << primeros + ultimos << endl;

    return 0;
}

Ejercicio 4 - Guía 4

Un postulante a un empleo realiza un test de capacitación. Se ingresa por teclado la cantidad total de preguntas realizadas y la cantidad de preguntas contestadas correctamente.

Desarrollar un programa que informe el nivel obtenido, sabiendo que:

  • Nivel máximo: Porcentaje >= 90.
  • Nivel medio: Porcentaje >= 75% y < 90%.
  • Nivel regular: Porcentaje >= 50% y < 75%.
  • Fuera de nivel: Porcentaje < 50%.

Solución: Ejercicio 4


#include <iostream>
using namespace std;

int main() {
    int total, correctas;
    cout << "Total de preguntas: ";
    cin >> total;
    cout << "Respuestas correctas: ";
    cin >> correctas;

    float porcentaje = (correctas * 100.0) / total;

    if (porcentaje >= 90)
        cout << "Nivel máximo" << endl;
    else if (porcentaje >= 75)
        cout << "Nivel medio" << endl;
    else if (porcentaje >= 50)
        cout << "Nivel regular" << endl;
    else
        cout << "Fuera de nivel" << endl;

    return 0;
}

Ejercicio 7 - Guía 4

Diseñe un algoritmo que lea un número de tres cifras y determine si es o no capicúa. Un número es capicúa si “se lee igual al derecho y al revés”.

Ejemplo:

  • Si se ingresa 383, indica que es capicúa.
  • Si se ingresa 473, indica que no es capicúa.

Solución: Ejercicio 7


#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "Ingrese un número de tres cifras: ";
    cin >> num;

    int centena = num / 100;
    int unidad  = num % 10;

    if (centena == unidad)
        cout << num << " es capicúa" << endl;
    else
        cout << num << " no es capicúa" << endl;

    return 0;
}

Ejercicio 6 - Guía 4

El cliente de un supermercado puede pagar con:

  • Tarjeta de débito ('d', 'D')
  • Billetera Santa Fe ('b', 'B')
  • Tarjeta de crédito ('c', 'C') en 1 o 3 cuotas.

Condiciones:

  • Crédito: En 1 cuota no hay recargo. En 3 cuotas hay un 10% de recargo.
  • Débito: Si supera $3000, hay un 15% de descuento. Si no, ningún cambio.
  • Billetera SF: Siempre 30% de descuento.

Solución: Ejercicio 6


#include <iostream>
using namespace std;

int main() {
    float monto, total;
    char pago;
    cout << "Monto: "; cin >> monto;
    cout << "Pago (C/D/B): "; cin >> pago;

    if (pago == 'c' || pago == 'C') {
        int cuotas;
        cout << "Cuotas (1 o 3): "; cin >> cuotas;
        if (cuotas == 3) {
            total = monto * 1.10;
            cout << "Por cuota: $" << total / 3 << endl;
        } else {
            total = monto;
        }
    } else if (pago == 'd' || pago == 'D') {
        if (monto > 3000) {
            total = monto * 0.85;
        } else {
            total = monto;
        }
    } else if (pago == 'b' || pago == 'B') {
        total = monto * 0.70;
    }
    cout << "Total: $" << total << endl;
    return 0;
}

Ejercicio 5 - Guía 4

Descripción: Se solicita escribir el código para ingresar desde el teclado un número de día de la semana (1-7) y un número de mes (1-12), y que imprima el nombre del día de la semana y el del mes correspondientes.

Si el día o el mes ingresado es incorrecto deberá mostrarse un mensaje de error.


Ejemplo:

  • Ingresando 3, 8 → Martes y Agosto
  • Ingresando 12, 7 → Día inexistente.

Solución: Ejercicio 5


#include <iostream>
using namespace std;

int main() {
    int dia, mes;
    
    cout << "Dia de la semana (1-7): "; cin >> dia;
    cout << "Mes (1-12): "; cin >> mes;
    
    if (dia > 7 || dia < 1 || mes > 12 || mes < 1) {
        cout << "Dia inexistente" << endl;
    } else {
        cout << "Dia: ";
        switch(dia) {
            case 1: cout << "Lunes"; break;
            case 2: cout << "Martes"; break;
            case 3: cout << "Miercoles"; break;
            case 4: cout << "Jueves"; break;
            case 5: cout << "Viernes"; break;
            case 6: cout << "Sabado"; break;
            case 7: cout << "Domingo"; break;
        }
        
        switch(mes) {
            case 1: cout << " y Enero\\n"; break;
            case 2: cout << " y Febrero\\n"; break;
            case 3: cout << " y Marzo\\n"; break;
            case 4: cout << " y Abril\\n"; break;
            case 5: cout << " y Mayo\\n"; break;
            case 6: cout << " y Junio\\n"; break;
            case 7: cout << " y Julio\\n"; break;
            case 8: cout << " y Agosto\\n"; break;
            case 9: cout << " y Septiembre\\n"; break;
            case 10: cout << " y Octubre\\n"; break;
            case 11: cout << " y Noviembre\\n"; break;
            case 12: cout << " y Diciembre\\n"; break;
        }
    }
    return 0;
}

Ejercicio 9 - Guía 4

Escribir un programa que dado un número del 0 al 9, escriba en pantalla los números restantes hasta el 10.


Ejemplo:

  • Ingrese un número: 7

Solución: Ejercicio 9


#include <iostream>

        case 1: cout << "2\\t";
        case 2: cout << "3\\t";

Ejercicio 10 - Guía 4

Realizar un programa que permita calcular el perímetro de un triángulo. El programa mostrará un menú con las opciones Equilátero, Isósceles y Escaleno.

Si la opción es incorrecta, muestra un error. Si es correcta, solicitará los datos necesarios (deben ser positivos) y calculará el perímetro.

Menú:

  1. Equilatero
  2. Isósceles
  3. Escaleno

Solución: Ejercicio 10


#include <iostream>
using namespace std;

int main() {
    int tipo;
    float l1, l2, l3, p = 0;

    cout << "1- Equilatero\\n2- Isoceles\\n3- Escaleno\\nTipo: ";
    cin >> tipo;

    if (tipo < 1 || tipo > 3) {
        cout << "Opcion invalida." << endl;
    } else {
        switch (tipo) {
            case 1:
                cout << "Lado: "; cin >> l1;
                if (l1 <= 0) cout << "El valor debe ser positivo." << endl;
                else p = l1 * 3;
                break;
            case 2:
                cout << "Lado repetido: "; cin >> l1;
                cout << "Lado distinto: "; cin >> l2;
                if (l1 <= 0 || l2 <= 0) cout << "Debe ser positivo." << endl;
                else p = l1 * 2 + l2;
                break;
            case 3:
                cout << "Lado 1: "; cin >> l1;
                cout << "Lado 2: "; cin >> l2;
                cout << "Lado 3: "; cin >> l3;
                if (l1 <= 0 || l2 <= 0 || l3 <= 0) cout << "Positivos." << endl;
                else p = l1 + l2 + l3;
                break;
        }
        
        if (p > 0) cout << "Perimetro: " << p << endl;
    }
    return 0;
}
